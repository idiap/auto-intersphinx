============================================================
 Automatically Links Package Documentation from Their Names
============================================================

.. todolist::

This package contains a Sphinx_ plugin that can fill intersphinx_ mappings
based on package names.  It simplifies the use of that plugin by removing the
need of knowing URLs for various API catologs you may want to cross-reference.
